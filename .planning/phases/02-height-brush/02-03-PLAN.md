```gdscript

```

Known gotcha: `generate_terrain()` does `mesh_instance = MeshInstance3D.new(); add_child(mesh_instance)` without removing the previous mesh_instance. A repeated call would leak meshes. `rebuild_mesh()` must remove the old one first. Also need to handle collision shape and props — check lines ~486 and ~595 of terrain_chunk.gd during execution to confirm whether generate_terrain re-creates them or if they must be torn down too.

Add to `game/terrain_chunk.gd`:

```gdscript
# Public rebuild used by the height brush. Tears down the previous mesh/collision
# (mirroring whatever set_lod() would do) and re-runs generate_terrain() which
# will re-apply all shader params (D-10).
func rebuild_mesh() -> void:
    # Remove stale mesh instance created by previous generate_terrain()
    if mesh_instance and is_instance_valid(mesh_instance):
        mesh_instance.queue_free()
        mesh_instance = null
    # Rebuild (generate_terrain recreates mesh, collision for LOD0-1, shader params)
    generate_terrain()
```

If during read you find that `set_lod()` already performs the same teardown dance, follow the same sequence it uses (including any collision/prop teardown). If `set_lod` only calls `generate_terrain()` directly without removing the mesh, then `rebuild_mesh()` is the first place this is done correctly — note that in the SUMMARY.

Do NOT touch trees/props in `rebuild_mesh()` — brush rebuilds happen too frequently to re-spawn props, and prop placement uses `get_height_at_world_pos()` which already accounts for offsets, so existing props will sit at the wrong height after a stroke but that's accepted for Phase 2 (props repositioning is a Phase 3+ concern — document in SUMMARY as a known limitation).

```gdscript
func _flush_dirty_chunks() -> void:
    if _frame_dirty.is_empty() or terrain_manager == null:
        return
    for coord in _frame_dirty.keys():
        if not terrain_manager.chunks.has(coord):
            continue
        var chunk = terrain_manager.chunks[coord]
        if chunk and chunk.has_method("rebuild_mesh"):
            chunk.rebuild_mesh()
    _frame_dirty.clear()
```
